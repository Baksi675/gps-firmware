enum MODULES_te

Identifies a subsystem for use in logging and CLI output.

Each enumerator corresponds to one subsystem. The numeric value is used as an index into modules_names to look up the subsystem's name string.

Note
Enumerators must remain contiguous and in the same order as the entries in modules_names to keep the index mapping valid.
