Detailed Description

Configuration structure for initializing a GPIO pin.

Passed to gpio_init. Fields that are irrelevant for the selected mode are ignored:

  • output_type, output_speed are only relevant for output and alternate function modes.
  • alternate_function is only relevant for GPIO_MODE_ALTERNATE_FUNCTION.
  • interrupt_trigger is only relevant for GPIO_MODE_INTERRUPT.
