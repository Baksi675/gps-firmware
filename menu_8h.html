Detailed Description

Menu module public API.

Author
github.com/Baksi675

This module provides an OLED display menu system backed by the SSD1309 driver. Two menu types are supported:

  • Selectable (MENU_TYPE_SELECTABLE): a scrollable list of static string options. The currently highlighted option can be retrieved via menu_get_selected_option.
  • Data view (MENU_TYPE_DATA_VIEW): a scrollable list that interleaves static option labels with live values fetched at render time via a MENU_CFG_ts::get_value_fn callback.

Both types support a navigation history stack via menu_set_prev_menu and menu_get_prev_menu, allowing a back-navigation pattern.

Typical usage:

  • Initialize the subsystem using menu_init_subsys
  • Populate a MENU_CFG_ts and register a handle with menu_init_handle
  • Start the subsystem using menu_start_subsys
  • Scroll with menu_scroll and render with menu_run_handle
