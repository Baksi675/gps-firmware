◆ GPIO_MODE_te

enum GPIO_MODE_te

GPIO pin operating mode.

  • GPIO_MODE_INPUT and GPIO_MODE_OUTPUT write to MODER.
  • GPIO_MODE_ALTERNATE_FUNCTION enables the AFR registers.
  • GPIO_MODE_ANALOG disables the digital buffer (for ADC).
  • GPIO_MODE_INTERRUPT configures SYSCFG EXTICR, EXTI IMR/RTSR/FTSR, and the NVIC rather than MODER.
Enumerator
GPIO_MODE_INPUT 

Digital input mode.

GPIO_MODE_OUTPUT 

Digital output mode.

GPIO_MODE_ALTERNATE_FUNCTION 

Alternate function mode (peripheral routing).

GPIO_MODE_ANALOG 

Analog mode (digital buffer disabled).

GPIO_MODE_INTERRUPT 

External interrupt mode (EXTI + NVIC configured).
