Detailed Description

Data log module public API.

Author
github.com/Baksi675

This module implements periodic GPS data logging to a FAT filesystem (via FatFs) on an SD card. Each log entry records time, latitude, longitude, orthometric height, movement speed, and movement direction as retrieved from the NEO-6 GPS module.

Typical usage:

  • Initialize the subsystem using datalog_init_subsys
  • Configure a log using DATALOG_CFG_ts
  • Register a log handle using datalog_init_handle
  • Start the subsystem using datalog_start_subsys
  • Periodically call datalog_run_handle to append log entries
